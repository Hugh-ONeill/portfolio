projects

Crystal Battle

AlphaZero-style battle agent for Gen 2 OU. The search core is Monte Carlo Tree Search guided by neural-network policy priors (PUCT selection), backed by expectiminimax with alpha-beta pruning and a custom, matchup-aware evaluation at the leaves. Behind it runs a full training pipeline: self-play generation, human-replay scraping (1,962 ladder games), a 579-dimensional per-state feature extractor, and policy/value nets trained in PyTorch then exported to ONNX for fast Rust-side inference. The trained agent plays live on Pokemon Showdown over a websocket connection.

Rust Python PyTorch ONNX MCTS

job-scan

Daily, resume-matched job scanner that runs itself on a timer and costs nothing to operate. Scrapes fresh postings from company ATS boards (Greenhouse, Lever, Ashby) and remote aggregators (HN "Who is hiring?", RemoteOK), drops obvious misses with cheap title/location pre-filters, then scores each survivor 0–100 against my resume with a local LLM. The model rates four independent dimensions — the final number is computed deterministically in Python, so the ranking is transparent and tunable rather than a black-box vibe. Dedupes against SQLite and delivers a ranked Markdown digest plus a Flask dashboard with application-status tracking. Runs unattended via systemd timers, no API keys.

Python Flask Ollama / Gemma SQLite systemd

grounded-rag in progress

Local, citation-grounded question answering over any document corpus. Ask a question, get an answer that quotes its sources with inline [1][2] citations — and that says "I don't know" when the retrieved context doesn't support an answer. Documents are chunked with overlap, embedded via Ollama, and stored in Postgres + pgvector; queries run hybrid retrieval (vector + keyword) with a rerank pass before a local Gemma model generates the streamed, cited response. Built around an evaluation harness that actually measures retrieval hit-rate@k and answer faithfulness — most RAG demos wire up libraries and never check whether retrieval works. Runs entirely locally on a single GPU; FastAPI backend, React/TS frontend, containerized with Docker and a k8s manifest.

Python FastAPI React / TS Postgres + pgvector Ollama Docker / k8s
